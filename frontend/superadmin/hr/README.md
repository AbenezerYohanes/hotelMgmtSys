# Superadmin HR (Frontend)

This folder contains simple Next.js components and a page under `/hr` for the Superadmin HR dashboard and employee list.

Usage

- Ensure `NEXT_PUBLIC_API_URL` points to the backend API (e.g. `http://localhost:4000/api`).
- Run Next app in `frontend/superadmin`: `npm run dev`.
